本项目是中山大学数学学院每日一题的模板, 使用方法为:
\begin{lstlisting}
\begin{daily}[类型][日期][星级][标题]
	正文
\end{daily}
\end{lstlisting}
其中类型如下:
\begin{quote}
	\makeatletter
	\newcommand\TestColour[1]{{\fboxsep0pt\fbox{\colorbox{#1}{\phantom{XX}}}}}
	\makeatother
	\linespread{1}\selectfont
	\ExplSyntaxOn
	\begin{itemize}
		\clist_map_inline:Nn \c_math_styles_clist {
		\item[\TestColour{#1color}] \texttt{#1}
			}
	\end{itemize}
	\ExplSyntaxOff
\end{quote}
\begin{daily}[类型][日期][星级][标题]
	正文
\end{daily}
\begin{daily}[theorem][theorems][theoremv][theoremt]
	正文
\end{daily}
